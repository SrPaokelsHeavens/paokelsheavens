# Heaven's Path Web - srpaokelsheavens.eu.org

## Visión del Proyecto
Una plataforma web minimalista con estética **Xianxia / Wuxia / Dark Fantasy** diseñada para ser el hub central de la comunidad "Sr. Paokel's Heavens". La web servirá como biblioteca de contenidos, centro de noticias y portal de comunidad/donaciones.

## Concepto Estético: "Tinta y Vacío"
- **Dark Fantasy:** Sombras profundas, texturas de papel antiguo, elementos visuales que evocan peligro y misticismo.
- **Xianxia/Wuxia:** Tipografía elegante, iconografía de espadas, sectas y pergaminos.
- **Minimalismo:** Interfaz limpia que prioriza el contenido y la facilidad de navegación (UX).
