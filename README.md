# Heaven's Path Web – English-First Chronicle Hub

## Project Vision
Sr. Paokel’s Heavens is a minimalist **Xianxia / Wuxia / Dark Fantasy** portal focused exclusively on English content. The site acts as the canonical archive for:

- Immersive landing experience with the Celestial Carousel.
- Imperial Library (novels, volumes, and rich chapter reader).
- Dao Table recognition wall.
- Donations onboarding and community touchpoints.

## Design Principles – "Ink & Void"
- **Atmosphere:** Deep blacks, parchment overlays, and gold accents for a ritualistic tone.
- **Typography:** Serif headlines for lore, modern sans for system UI.
- **Focus:** Zero language toggles, zero alternate locales—every surface is authored in English to avoid regressions from the retired i18n system.

## Current Site Structure
- **Home:** Hero speech + Celestial Carousel fed by Markdown collections.
- **Library:** Novel grid, per-novel index, and chapter reader with TTS/reading-progress.
- **Dao Table:** Markdown-driven tiers (Celestial Body -> Celestial Sovereign) feeding both the Dao Table page and the hero carousel.
- **Donations:** PayPal / Ko-fi embeds plus CTA for future supporters.
- **Community:** Signal beacon with official dispatch routes (news, ledgers, donations) plus the direct archivist mailbox—no live chat or external platforms.

For contribution guidelines and automation notes, review the updated documents inside docs/. Every change should reinforce the English-only constraint and avoid reintroducing multilingual scripts.
